# 🗃️ | StockWise.
**A stock management REST API.**
# 📥 Installation
**Clone the repository:**
```bash
git clone https://github.com/Furyforev3r/StockWise.git
```
**Install dependencies:**
```bash
npm install
```
# 🖥️ How to Run
**Run the command:**
```bash
npm run dev
```
